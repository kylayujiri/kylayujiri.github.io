Movie Tracker Web App

I made a web app for tracking movies using Node.js, Express.js, Passport.js, MongoDB, Handlebars, and Bootstrap for the final project of my class, Applied Internet Technology (Fall 2022).

With Movie Tracker, a user can register, add movies, add how many times they've seen a movie, favorite movies, and view other users' profiles. For this project, I had a limited time frame so I decided to focus on functionality over aesthetics. Bootstrap made making a simple layout easy.
